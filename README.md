# 📊 Google Play Store Analytics Dashboard

## 🚀 Project Overview
This project analyzes the Google Play Store dataset using Python and Plotly to create interactive dashboards. It includes 6 advanced visualization tasks implemented as part of a data analytics internship.

---

## 📁 Dataset
- Google Play Store Dataset
- Contains app details such as installs, reviews, ratings, size, category, etc.

---

## 🔧 Technologies Used
- Python
- Pandas
- Plotly
- Google Colab

---

## 📊 Tasks Implemented

### 🔹 Task 1: Bubble Chart
Analyzed relationship between app size, rating, and installs.

### 🔹 Task 2: Choropleth Map
Visualized installs across top categories using geographic representation.

### 🔹 Task 3: Time Series
Trend of installs over time with growth highlighting.

### 🔹 Task 4: Stacked Area Chart
Cumulative installs across categories.

### 🔹 Task 5: Grouped Bar Chart
Comparison of average ratings and reviews.

### 🔹 Task 6: Dual Axis Chart
Comparison of installs and revenue (Free vs Paid apps).

---

## ⚙️ Key Features
- Advanced filtering conditions
- Time-restricted dashboards (IST-based)
- Multi-language category translation
- Growth trend highlighting
- Revenue estimation
